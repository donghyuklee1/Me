# Donghyuk Lee - Personal Website

Personal website showcasing research, projects, and professional information.

## Features

- Dark/Light mode toggle
- Research sections with detailed information
- Photo gallery with captions
- Status updates section
- Responsive design
- Color theme customization

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

## Setup

Simply open `index.html` in a web browser or serve using a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`
